# 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/MobinDevN/mobinnz-portfolio.git
```

Move into project directory

```bash
cd mobinnz-portfolio
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

Open

```
http://localhost:5173
```

---

# 📦 Production Build

Build the project

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
